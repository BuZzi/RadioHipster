- Erreur 404 (problème de routing) ? => Décommenter la ligne contenant le module 'mod_rewrite' dans le httpd.conf du serveur apache.
- Problème d'upload en local ? => mettre le 'safe_mode' en 'On' dans le php.ini du serveur.
